준비물 2 : git을 다운 받고 설치해주세요. git은 소스코드를 사진찍는 프로그램입니다.
보통, 소스코드를 서버에 업로드하거나 할 때, 파일을 그대로 보내기보단, 파일을 git으로 사진 찍어서 가릴 건 가리고 보정할 건 보정해서 보냅니다.

준비물 3 : github에 가입해주세요. (https://github.com)
프로그래밍 업계의 인스타그램 같은 곳입니다. git이 사진 찍는 프로그램이랬잖아요. 인싸들이 사진을 인스타그램에 올리듯이, 프로그래머는 git을 github에 올립니다.
저희는 코드를 git으로 사진 찍어서, github에 올리고, azure는 github에 올려진 코드를 읽어서 서버를 구축해줄 거예요.


준비물 4 : github desktop을 설치해주세요. (https://desktop.github.com)

git은 원래 git bash라는 멍멍 같은 프로그램으로 다뤄야 하는데, github desktop을 쓰면 훨씬 편하게 git을 github에 업로드 할 수 있습니다.
